# Quick Start Guide

## Terminal Auto-Navigation

Your terminal is configured to automatically navigate to the qortex project directory when you open a new terminal session.

This means you can directly run:
```bash
npm run dev
```

Without needing to `cd` into the directory first!

## Quick Commands

- **Start development server**: `npm run dev`
- **Build project**: `npm run build`
- **Run linter**: `npm run lint`
- **Navigate to qortex** (if you're elsewhere): `qortex`

## Configuration

This auto-navigation is set up in your `~/.bashrc` file. If you need to disable it, you can comment out the auto-cd section in that file.
